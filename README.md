Codeforces Progress Dashboard

A modern web dashboard that analyzes any Codeforces handle and provides detailed insights about problem solving performance.

Live Demo: https://marouf-jk.github.io/codeforces-users-progress/

Features:

Daily solved problems tracking
Monthly activity visualization
Rating & tags breakdown
Attempts before solving problems
Consistency and streak analysis
Best training day detection

Tech Stack:

HTML
CSS
JavaScript
Codeforces Public API

How It Works:

The dashboard fetches public submissions using the Codeforces API and processes the data directly in the browser to generate analytics.
